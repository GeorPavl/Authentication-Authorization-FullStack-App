1) npm init 

2) npm install --save angular@1.8.0

3) npm install --save @uirouter/angularjs@1.0.29

4) npm install --save angular-route@1.8.0

5) npm install --save-dev gulp@3.9.1

6) npm install --save-dev gulp-concat@2.6.1

7) npm install --save-dev browser-sync@2.18.2

8) Δημιουργώ φακέλους
    - dist
    - node_modules
    - src
        - app 
        - assets 
        - messages
        - vendor
    - gulpfile.js

9) Δημιουργώ στο vendor τα αρχεία για το gulp (styles.json, scripts.json) και εισάγω τα path για τα modules 

10) Δημιουργώ το app.js και αρχικοποιώ το πρώτο state.

11) Δημιουργώ έναν controller και ένα view για το home 

12) Εγκαθιστώ το bootstrap, Δημιουργώ φακέλους:
        - vendor
            - bootstrap
            - jquery
            - popper.js

13) npm install angular-resource@1.8.0 (και προσθήκη στα scripts και στο app.js)

14) Δημιουργώ τα αρχεία .consts .factory .service και τα συνδέω με controller

15) npm install angular-animate@1.8.0  (και προσθήκη στα scripts και στο app.js)

16) Δημιουργώ το αρχείο animations.css στον φάκελο css 

17) npm install angular-messages@1.8.0 (και προσθήκη στα scripts και στο app.js)

18) Εφαρμογή παραδείγματος για messages από https://docs.angularjs.org/api/ngMessages/directive/ngMessages

19) npm install --save chart.js@2.7.2 (dependenct για angular-chart)
